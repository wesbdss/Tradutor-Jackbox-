JACKBOX PARTY 6: Push the Button Tradutor

============================= Como Traduzir? ============================
1. Extraia o conteúdo da pasta "PushtheButton" para a pasta raiz do game.
2. Abra o executavel "jackboxproject.exe".
3. Click em traduzir.
4. Aguarde o processo ser finalizado.
5. Abra o Jogo e divirta-se!

================================= Pastas ================================
A pasta "ArquivosTraduzidos" contém os .txt para a tradução.
A pasta "ObtemFrases" terá o conteúdo retirado do game para a tradução.
